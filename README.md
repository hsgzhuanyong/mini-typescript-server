# 一个基于typescript + express 的小型服务器

## 如何使用

> 安装依赖文件

```shell
  yarn install
```

> 运行开发环境

```shell
  yarn dev
```

> 打包上线
```shell
  yarn build
```

目录解析

    src                     主目录文件
      config                项目配置文件
        index.ts            项目配置文件
      types                 类型定义
        config.ts           项目配置文件类型定义
      app.ts                程序入口文件
    package.json
    tsconfig.json
